== Description ==

Include java applets inside psots and pages via shortcode, supports multiple files and parameters.

== Installation ==

Use the shortcode:
[sketch name="name" listostuff="/path/file1.jar,/path/file2.jar,/path/file3.jar" width="400" height="400" param="parameter1:value1;parameter2:value2;parameter3:value3"]<br/>
'name' is the class name<br/>
